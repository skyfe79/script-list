# @skyfe79/script-list

A fast Rust CLI tool to list npm scripts from package.json, distributed via npm.

## Installation

```bash
npm install -g @skyfe79/script-list
```

## Usage

```bash
# List all scripts
sl

# Filter scripts
sl -f build

# Different formats
sl -F json
```

## Binary Installation

This package automatically downloads the appropriate binary for your platform during installation.

Supported platforms:
- macOS (ARM64, x64)
- Linux (x64)
- Windows (x64)
